### 三、功能描述
系统分为（身份）：普通用户（学生）、教师、组长、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 课程
5. 作业
6. 作业互评
7. 个人中心


**教师功能：**
1. 登录
2. 注册
3. 个人中心（修改密码、个人信息）
4. 小组信息管理
5. 课程管理
6. 作业管理
7. 作业互评管理
8. 作业评分管理

**组长功能：**
1. 登录
2. 注册
3. 个人中心（修改密码、个人信息）
4. 课程
5. 作业
6. 作业互评
7. 作业评分





**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 学生管理
4. 教师管理
5. 组长管理
6. 小组信息管理
7. 课程管理
8. 作业管理
9. 作业互评管理
10. 作业评分管理
